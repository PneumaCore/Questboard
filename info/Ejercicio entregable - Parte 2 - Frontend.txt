Ya tienes la API REST funcionando como un reloj suizo, ¡bien hecho!
Ahora es momento de dar un paso más: crear el Frontend que le pondrá cara a
todo ese músculo técnico. Porque, seamos sinceros, el código más bello del
mundo no impresiona a nadie si no tiene una interfaz que lo haga brillar.
¿Qué haremos?
Vamos a crear una app en Vue 3 que se conecte con tu API y permita a los
usuarios gestionar su backlog de videojuegos con estilo. Esta será la
herramienta que los gamers necesitan para organizar su caos y cumplir la
misión de pasar sus pendientes.

juegos.

Resumen funcionalidades que debe tener
-

Conexión con la API:
- Crear, listar, editar y borrar juegos.
- Ordenar y ﬁltrar resultados.
- Marcar juegos como completados (registrando la fecha automáticamente).

-

Interfaz limpia y funcional:
- Usar componentes reutilizables para formularios, listas y tarjetas.
- Estilo básico (puedes usar Tailwind CSS o Bootstrap para acelerar).

Sprint 2: Frontend para múltiples gamers (continuación)

Resumen funcionalidades que debe tener
-

Autenticación:
- Implementar el ﬂujo de registro y login.
- Guardar el token de autenticación (por ejemplo, en localStorage).
Protección de rutas:
- Restringir acceso a la lista de juegos, creación, edición, etc., solo a usuarios
autenticados.
Manejo de sesiones:
- Cerrar sesión de forma limpia.
- Mostrar la información correcta según el usuario logueado.
